About
Ocean Escape is a fun browser game where you control a sea turtle 🐢, collect snails 🐌, avoid sharks 🦈, and survive as long as possible.

Features
🐢 Turtle Player
🦈 Shark Enemies
🐌 Collectible Snails
⏱️ Survival Timer
🏆 Score System
🌊 Ocean Theme
